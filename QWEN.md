# Калькулятор PWA — Контекст проекта

## Обзор проекта

**Калькулятор** — это прогрессивное веб-приложение (PWA), реализующее функционал стандартного калькулятора в стиле iOS. Приложение работает офлайн, поддерживает установку на устройства и полноэкранный режим.

### Основные возможности
- Базовые арифметические операции (+, −, ×, ÷)
- Дополнительные функции: ± (смена знака), % (проценты)
- Поддержка клавиатурного ввода
- Адаптивный дизайн для мобильных и десктопных устройств
- PWA-функции: офлайн-режим, установка на устройство, полноэкранный режим
- Автоматическое масштабирование шрифта результата

## Структура проекта

```
Калькулятор/
├── index.html          # Главная страница приложения
├── styles.css          # Стили с адаптивным дизайном
├── script.js           # Логика калькулятора (класс Calculator)
├── manifest.json       # PWA-манифест для установки приложения
├── sw.js               # Service Worker для офлайн-режима
├── icon-*.png          # Иконки различных размеров (72–512px)
├── icon.svg            # SVG-исходник иконки
├── generate.js         # Node.js-скрипт генерации иконок (sharp)
├── generate-icons.html # Браузерная версия генератора иконок
└── package.json        # Зависимости и скрипты
```

## Технологии

- **Frontend:** Vanilla JavaScript, CSS3, HTML5
- **PWA:** Service Worker, Web App Manifest
- **Иконки:** Sharp (Node.js) для генерации PNG из SVG

## Сборка и запуск

### Установка зависимостей
```bash
npm install
```

### Генерация иконок
```bash
npm run generate
```

### Запуск приложения
Откройте `index.html` в браузере или используйте локальный сервер:
```bash
npx serve .
# или
npx http-server -p 8080
```

### Требования для PWA
- HTTPS (или localhost для разработки)
- Все иконки должны присутствовать в корне проекта

## Архитектура

### Класс `Calculator` (`script.js`)
```javascript
{
    currentValue: string,      // Текущее значение на дисплее
    previousValue: number|null,// Предыдущее значение для операций
    operator: string|null,     // Текущая операция
    waitingForOperand: bool,   // Ожидание следующего операнда
    expression: string         // Строка выражения для отображения
}
```

**Основные методы:**
- `inputNumber(num)` — ввод цифры
- `inputOperator(op)` — ввод операции
- `calculate()` — вычисление результата
- `clear()` — сброс
- `toggleSign()` — смена знака
- `percent()` — процент
- `inputDecimal()` — ввод десятичной точки
- `handleKeyboard(e)` — обработка клавиатуры

### Service Worker (`sw.js`)
- **Стратегия кэширования:** Cache First с fallback на Network
- **Версия кэша:** `calculator-v2`
- **Кэшируемые ресурсы:** index.html, styles.css, script.js, manifest.json, иконки

### PWA Manifest (`manifest.json`)
- **display:** fullscreen (полноэкранный режим)
- **orientation:** any (любая ориентация)
- **theme_color:** #000000 (чёрная тема)

## Стилевые особенности

### Цветовая схема (в стиле iOS)
- **Фон:** #000000 (чёрный)
- **Цифры:** #333333 (тёмно-серый)
- **Операторы:** #ff9f0a (оранжевый)
- **Функции:** #a5a5a5 (светло-серый)
- **Равно:** #ff9f0a (оранжевый)

### Адаптивность
- `clamp()` для размеров шрифтов и отступов
- Grid-layout для кнопок (4 колонки)
- Поддержка safe-area-inset для устройств с челкой
- Медиа-запросы для планшетов и пейзажной ориентации

## Конвенции разработки

### Код
- Классы и функции на ES6+
- События через `addEventListener`
- data-атрибуты для связки UI и логики
- Локализация чисел через `toLocaleString('ru-RU')`

### Ограничения
- Максимум 12 цифр в числе
- Автоматический переход в экспоненциальную форму при |value| ≥ 1e9
- Деление на ноль возвращает 0

### Обработка клавиатуры
| Клавиша | Действие |
|---------|----------|
| 0–9 | Ввод цифры |
| +, -, *, / | Операция |
| Enter, = | Вычислить |
| Escape, C | Сброс |
| Backspace | Удалить символ |
| . | Десятичная точка |
| % | Процент |

## Примечания

- Приложение не требует сборки — работает напрямую из статических файлов
- Для PWA-функционала необходим HTTPS (кроме localhost)
- Fullscreen режим активируется автоматически при загрузке и по первому клику
